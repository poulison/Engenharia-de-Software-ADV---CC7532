## Descrição do Projeto

Uma breve descrição sobre o sistema que faremos esse semestre, ele está detro do dominio da saude e busca melhorar os habitos cotidianos dos usuarios tendo o publico alvo as pessoas que buscam uma melhora nos seus habitos. O sistema tera as funcionalidades principais dele sendo Cadastro, Login, Catálogo de Exercícios Físicos, Informações nutricionais.
